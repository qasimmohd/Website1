Game Rental Information

Rental Periods

Games can be rented for different time periods:

  • Hourly (during cafe visit)
  • Daily (take home for 24 hours)
  • Weekend (Friday to Sunday)
  • Weekly (7 days)

Game Components

All our games are regularly checked to ensure they include all necessary components. Each game includes:

  • Complete set of game pieces
  • Original instruction manual
  • Game box in good condition

Condition Policy

We maintain our games in excellent condition. When renting:

  • Games are inspected before and after rental
  • Damage fees apply for missing or damaged components
  • Security deposit required for take-home rentals
